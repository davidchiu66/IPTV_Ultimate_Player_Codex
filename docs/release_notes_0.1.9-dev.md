# Release Notes - 0.1.9-dev

## 本次重点

本版本聚焦播放操作体验补强，新增常用键盘快捷键，并修复切换资源后音量设置不保持的问题。

## 新增功能

### 播放器键盘快捷键

新增两个播放器级快捷键：

1. `空格键`：播放 / 暂停切换。
2. `回车键`：窗口 / 全屏切换，同时支持主键盘 `Return` 和小键盘 `Enter`。

适用范围：

1. 直播播放。
2. 本地视频 / 音频播放。
3. 本地媒体播放结束后，按 `空格键` 可复用现有逻辑重新播放。

为避免误操作，快捷键不会抢占搜索框、输入框、下拉框、列表、按钮、滑条和弹窗中的正常键盘交互。

## 修复问题

### 音量设置切换资源后不保持

修复音量调节只对当前媒体有效的问题。现在用户调整音量或静音状态后：

1. 切换到下一个视频、音频或直播时会继续沿用当前音量。
2. 停止播放后再设置音量或静音，下一次播放也会继承该状态。
3. 新建或重载 libmpv 播放会话时，会自动恢复最近一次音量和静音状态。

## 技术调整

1. 扩展主窗口键盘事件归属判断，兼容 libmpv native 播放窗口获得焦点的场景。
2. 在主窗口事件过滤器和 `keyPressEvent` 中复用同一套播放器快捷键处理逻辑。
3. 为 mpv 视频承载控件启用强焦点策略，提升播放区域接收键盘事件的稳定性。
4. 在 `MpvVideoWidget` 内维护用户偏好的音量和静音状态，并在每次 `loadfile` 前恢复。

## 验收建议

1. 播放直播时按 `空格键`，可暂停 / 继续播放。
2. 播放本地视频时按 `空格键`，可暂停 / 继续播放。
3. 窗口状态按 `回车键`，进入全屏；全屏状态再次按 `回车键`，恢复窗口。
4. 在搜索框、资源列表、频道列表、设置项或弹窗中按 `空格键` / `回车键`，不会误触发播放器快捷键。
5. 将音量调到非 100%，切换到另一个视频或直播后，音量仍保持为用户设置值。
6. 静音后切换资源，新资源仍保持静音状态。
