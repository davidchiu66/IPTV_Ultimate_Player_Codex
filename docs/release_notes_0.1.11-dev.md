# Release Notes - 0.1.11-dev

## 版本重点

本版本继续打磨播放体验、资源库交互和点击式面板操作模式，重点解决高 DPI、自绘界面、libmpv 原生视频层级、播放参数持久化和资源侧边面板可用性问题。

## 新增功能

### 面板点击滑入操作模式

系统设置新增“面板滑入操作模式”配置项：

1. 鼠标移动触发：保留原有边缘 hover 自动滑入模式。
2. 鼠标点击触发：通过屏幕边缘方向箭头手动滑入或滑出资源库、频道列表、播放列表、顶部条和控制面板。

点击模式下的边缘触发规则：

1. 未播放时显示左、右、上三个入口，分别控制资源库、播放列表和工具栏。
2. 播放中显示左、右、上、下四个入口，分别控制左侧面板、播放列表、播放器顶部条和播放器控制面板。
3. 直播上下文下点击左侧入口优先打开频道分组与频道列表；非直播或频道列表为空时打开资源库。
4. 面板滑入后方向箭头自动反向，点击可滑出。

### 播放态边缘箭头空闲隐藏

点击模式进入播放后，边缘箭头会根据鼠标活动自动呈现或隐藏：

1. 鼠标静止约 4.2 秒后自动隐藏四个方向箭头。
2. 鼠标再次移动、滚轮或点击后自动恢复显示。
3. 针对 libmpv 原生视频窗口增加光标轮询兜底，避免 Qt 收不到 MouseMove 事件时无法恢复箭头。

## 界面优化

### 边缘触发按钮视觉简化

1. 去除原来的玻璃背景方块和边框。
2. 保留纯白方向箭头，并增加轻微阴影和淡蓝高光。
3. 保留原点击热区大小，视觉更轻，但操作仍然容易命中。

### 面板自动隐藏策略

点击模式下，资源库、频道列表、详情、播放列表、设置和工具栏不再自动因为鼠标离开而滑出，避免用户正在操作时面板被收起。

## 修复问题

### libmpv 播放层遮挡边缘触发按钮

修复进入窗口播放或全屏播放后，左、右、上、下四个边缘触发按钮消失、无法点击的问题。

处理方式：

1. 边缘触发按钮改为 `Qt.Tool + Frameless + WindowStaysOnTop` 轻量浮动窗口。
2. 使用 `video_stack_host.mapToGlobal()` 按全局坐标定位，兼容窗口播放和全屏播放。
3. 窗口移动、尺寸变化、全屏切换和播放状态变化后自动重定位并提升层级。

### 点击模式左侧入口打开错误面板

修复直播状态下点击左侧箭头错误打开资源库的问题。

现在左侧入口复用统一上下文判断：

1. 有可用频道列表且当前为直播上下文时，打开频道分组与频道列表。
2. 本地资源、非频道播放或频道列表为空时，打开资源库。

### 播放状态同步问题

1. `PlayerPanel` 新增播放 UI 状态变化信号，在开始加载、播放成功和停止时通知主窗口刷新边缘入口。
2. 修复 mpv 控件已存在时重新播放未切回视频栈的问题。
3. 边缘按钮刷新不再依赖可能残留的 `current_channel`，改为根据播放器实际状态判断。

### 音量和倍速跨文件保持

修复音量、静音和播放速率仅对当前文件有效的问题。手动切换、本地播放列表自动连播和相邻资源切换时，未再次修改前会继续沿用用户当前设置。

### EPG 下载告警处理

优化 `.xml.gz` 节目单下载和解析提示，对“扩展名为 gzip 但实际不是 gzip”的响应进行更清晰的识别和告警，降低后台异常噪声。

### Qt 控件层级告警

继续规避部分 Qt 子控件与原生窗口组合时的层级告警，减少类似以下日志：

```text
QWidgetWindow(..., name="appleTVControlBarWindow") must be a top level window
```

## 技术调整

1. 新增 `ui/edge_trigger_buttons.py`，封装点击模式边缘方向箭头。
2. 新增 `utils/panel_interaction_settings.py`，持久化面板滑入操作模式。
3. `BaseOverlay` 和 `ToolbarOverlay` 增加自动隐藏开关。
4. `PlayerPanel` 增加点击模式控制条手动显示/隐藏能力。
5. `MainWindow` 增加边缘按钮定位、层级刷新、空闲隐藏和播放上下文路由逻辑。

## 验收建议

1. 设置为“鼠标移动触发”，确认原有边缘 hover 滑入逻辑仍正常。
2. 设置为“鼠标点击触发”，未播放时确认左、右、上三个箭头可控制资源库、播放列表和工具栏。
3. 进入直播播放后，确认四个箭头显示，并且左侧打开频道分组与频道列表。
4. 进入本地视频播放后，确认左侧打开资源库，右侧打开播放列表，上下箭头控制顶部条和控制面板。
5. 播放中静止鼠标约 4.2 秒，确认四个箭头隐藏；移动鼠标后确认箭头恢复。
6. 窗口播放和全屏播放都验证一次，确认箭头不会被 libmpv 视频画面遮挡。
7. 调整音量和倍速后切换文件或自动连播，确认设置仍然生效。
