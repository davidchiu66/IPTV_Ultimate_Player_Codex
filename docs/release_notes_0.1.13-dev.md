# Release Notes - 0.1.13-dev

## 版本重点

本版本重点增强右侧播放列表的管理能力：播放列表现在支持添加文件、多选移除、手动排序、删除播放列表，以及刷新后保留用户自定义顺序。

同时修复 TXT 频道文件解析中的一个非标准元数据误判问题，避免裸 `ua=...` 行被识别成频道。

## 播放列表增强

### 播放列表删除入口

播放列表名称右侧现在保留 `+` 新建按钮，并新增 `-` 删除播放列表按钮。

1. `+` 用于新建播放列表。
2. `-` 用于删除当前播放列表。
3. 默认播放列表仍不可删除。
4. 删除播放列表只删除播放列表关系，不删除原始媒体文件。

### 添加文件

播放列表底部操作区新增“添加”按钮，可向当前播放列表添加一个或多个本地媒体文件。

1. 支持一次选择多个文件。
2. 只添加项目支持的本地媒体类型。
3. 新文件追加到当前播放列表末尾，不打乱已有顺序。
4. 已存在文件不会重复添加。
5. 曾经从播放列表“移除”的文件，手动添加后会恢复到列表末尾。
6. 手动添加的外部目录文件在刷新播放列表后仍会保留。

### 多选移除

原底部“删除”按钮改为“移除”，用于从当前播放列表移除选中条目。

1. 支持 `Ctrl` 多选不连续条目。
2. 支持 `Shift` 连续选择。
3. 选中条目会使用现有列表高亮样式。
4. 移除操作只从播放列表中移除条目，不删除磁盘上的原始媒体文件。
5. 被移除但磁盘仍存在的文件会记录到排除列表，刷新时不会自动扫回。

### 手动排序

播放列表新增一排排序按钮：

1. `置顶`
2. `上移`
3. `下移`
4. `置底`

排序支持单选和多选。多选移动时会保持选中条目之间的相对顺序。排序结果会写入 `playlists.json`，下次打开或重启后仍按最新顺序加载。

### 刷新保序

播放列表刷新逻辑现在会尽量保留用户整理过的顺序：

1. 已存在条目保持当前手动顺序。
2. 新扫描到的文件追加到末尾。
3. 手动添加的外部文件只要仍存在且类型受支持，就会继续保留。
4. 磁盘已删除或类型不再支持的文件会从列表中移除。
5. 用户手动移除的文件不会因为刷新被重新扫回。

## TXT 频道解析修复

修复 TXT 频道文件解析时，把非标准元数据行误解析成频道的问题。

例如以下内容现在会被识别为元数据行并跳过，不会出现在频道列表中：

```text
ua=Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML,like Gecko) Chrome/108.0.0.0 Safari/537.36
```

当前过滤范围包括：

1. `ua=...`
2. `user-agent=...`
3. `http-user-agent=...`
4. `referer=...`
5. `referrer=...`
6. `header=...`
7. `headers=...`

正常 `频道名,播放地址` 行不受影响。

## 技术调整

1. `backend/playlist_album_manager.py` 新增 `add_items()`、`remove_items()`、`move_items()`。
2. 播放列表新增 `excluded_paths`，用于记住被手动移除但磁盘仍存在的文件，避免刷新时重新扫回。
3. 播放列表刷新合并逻辑改为保留手动顺序、保留手动添加文件、追加新文件。
4. `ui/playlist_overlay.py` 开启 `QListWidget.ExtendedSelection`，支持多选高亮。
5. `ui/playlist_overlay.py` 新增添加、移除和排序相关信号。
6. `ui/main_window.py` 接入播放列表文件多选添加、移除确认、排序刷新和状态栏反馈。
7. `backend/playlist_manager.py` 增加 TXT 裸元数据行过滤。

## 验收建议

1. 打开右侧播放列表，确认播放列表名称右侧有 `+` 和 `-`。
2. 点击“添加”，一次选择多个本地媒体文件，确认条目追加到末尾。
3. 重复添加同一文件，确认不会重复出现。
4. 选中一个或多个条目点击“移除”，确认只从列表移除，磁盘文件仍存在。
5. 移除后点击“刷新”，确认移除的文件不会自动扫回。
6. 对已移除文件再次点击“添加”，确认能恢复到列表末尾。
7. 使用“置顶 / 上移 / 下移 / 置底”调整单个和多个选中条目，确认相对顺序正确。
8. 重启程序，确认播放列表顺序、添加结果和移除结果都能持久保存。
9. 加载包含裸 `ua=...` 行的 TXT 频道文件，确认该行不会显示为频道。
